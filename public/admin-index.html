Admin

Internal tools on the health server (not MCP). Use on trusted networks only.

  • Scheduled posts — SQLite queue: cancel pending rows or send now.
  • Draft posts — Approve, reject, or schedule rows from draft_posts.
  • Competitor insights — Public snapshot signals from competitor_signals (same JSON as MCP generate_competitor_report).
  • Post performance — Cached impressions / reach / engagement from post_insights (filled by MCP get_post_insights).
  • Supplier email upload — Drag-and-drop .eml / .msg / .zip for ingestion into supplier_updates.
